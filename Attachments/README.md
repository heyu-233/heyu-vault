# Attachments

Use this folder for small public assets used by notes.

## Rules

- Keep files small.
- Prefer public, non-sensitive images or diagrams.
- Put private files under `Attachments/private/`; that path is ignored by Git.
- Do not commit VM images, ISOs, archives, or private screenshots.
