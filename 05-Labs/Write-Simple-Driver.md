# Write Simple Driver

## Goal

Write or study a minimal Linux driver to understand module loading, device registration, and basic file operations.

## Steps

1. Build a minimal module.
2. Load and unload it.
3. Inspect dmesg output.
4. Add a simple file operation if appropriate.
