# openEuler Overview

openEuler is an open source operating system community under the OpenAtom ecosystem. For this vault, openEuler is treated as the practical operating-system target for Linux kernel learning, experiments, and contribution.

## Focus Areas

- Linux kernel and system software
- multi-architecture support
- cloud, server, edge, and embedded scenarios
- SIG-based open source collaboration
