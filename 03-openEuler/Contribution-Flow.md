# Contribution Flow

## Standard Flow

1. Find issue or candidate task.
2. Confirm whether the task is still available.
3. Fork repository.
4. Create topic branch.
5. Reproduce or understand expected behavior.
6. Implement minimal change.
7. Run tests and document results.
8. Submit PR with clear description.
9. Respond to review.
